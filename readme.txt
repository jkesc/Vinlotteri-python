Script which runs a wine lottery wheel of fortune
execute the GUI.py file to run the wheel of fortune with a GUI.
execute the wheel.py file to run the lottery with hard-coded values.
The Gui takes names and number of tickets for each participant. These values can be adjusted manually in the GUI if required
if the button "Run lottery" is activated, the wheel of fortune is executed, and the winner is deducted one ticket for the next run.
